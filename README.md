# Update-main.c
Tugas DPK (Program Data Base)
Cara Menjalankan Kode
pull kodenya di pc kamu lalu buka terminal di vs code dengan menjalankan perintah berikut

gcc main.c filedata.c -o main.exe
setelah menjalankan perintah tersebut file main.exe akan muncul. kemudian jalankan kode ini di terminal

./main.exe 1
untuk menjalankan main.exe 1 (Mata Kuliah)

./main.exe 2
untuk menjalankan main.exe 2 (Mahasiswa)

./main.exe 3
untuk menjalankan main.exe 3 (KRS)

jalankan satu per satu main.exe
